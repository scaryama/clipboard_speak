# 🗣️ Clipboard Speak (TTS)

[English](README.md) | 한국어

클립보드나 터미널 인자로 텍스트를 전달하면, 음성으로 즉시 읽어주는 도구입니다.

## 🚀 사용법

### 1. 설치
```bash
pip install -r requirements.txt
```

### 2. 실행
*   **클립보드 텍스트 읽기:**
    ```bash
    python speak.py
    ```
*   **터미널 입력 텍스트 읽기:**
    ```bash
    python speak.py "읽을 텍스트"
    ```

## ⚡ 주요 특징: 빠른 응답 시간
본 도구는 긴 텍스트를 문장 단위로 자동 분할하여 처리합니다.
- **병렬 처리 및 즉시 재생:** 텍스트 전체의 음성 합성을 기다리지 않고, 첫 문장의 오디오가 준비되는 즉시 재생을 시작합니다. 
- **지연 시간 최소화:** 첫 문장이 재생되는 동안 이어지는 문장들의 데이터 처리가 백그라운드에서 병렬로 진행되어, 끊김 없는 자연스러운 재생 환경을 제공합니다.
